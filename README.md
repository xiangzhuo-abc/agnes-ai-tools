# Agnes AI 生成工具

自然语言描述 → 自动识别视频/图片 → 一键生成，保存到指定文件夹。

## 文件

| 文件 | 说明 |
|------|------|
| `agnes-gui.py` | GUI 桌面工具，支持文生图/图生图/文生视频/图生视频/关键帧 |
| `agnes-tools.ps1` | PowerShell 命令行快捷工具 |
| `config.example.json` | 配置模板 |

## 环境

- Python 3.10+
- 可选：`pip install Pillow`（自动压缩大图，未安装也能用）

## 启动

```powershell
python agnes-gui.py
```

## 功能

- 5 种生成模式：文生图 / 图生图 / 文生视频 / 图生视频 / 关键帧
- 自然语言自动识别图片/视频意图，也可手动切换
- 时长：自动从提示词提取 或 手动 Spinbox 1-18 秒
- 参考图：URL 粘贴或本地文件，PIL 自动缩放压缩
- 图片尺寸：38 种（1K-4K × 比例 + 传统固定像素）
- 设置持久化到 `%USERPROFILE%\.agnes\config.json`

## 配置

首次启动会自动弹出设置窗口，填入你的 API Key 即可。

所有设置在 `%USERPROFILE%\.agnes\config.json`，**不要将含真实 Key 的 config.json 分享给他人**。

```json
{
  "api_key": "sk-你的APIKey",
  "output_dir": "D:\\Desktop\\Agnes生成",
  "image_size": "1024x1024",
  "video_duration": 5,
  "video_fps": 24
}
```

> ⚠️ **安全提示**：API Key 仅在本地存储，请勿将 `~/.agnes/config.json` 打包进分发包。
> 获取 API Key：https://platform.agnes-ai.com

## Agnes API 限制（官方 2026.06）

| 项目 | 限制 |
|------|------|
| 视频最大帧数 | ≤441，8n+1 |
| 视频最大时长 | ~18s（441÷24fps） |
| 视频日配额 | 500 秒/天 |
| 图片日配额 | 4,000 张/天 |
| RPM | 20 次/分钟 |
| 图片最大 | 4K（4096×4096） |
